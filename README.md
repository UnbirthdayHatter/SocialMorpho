# Social Morpho

Social Morpho is a custom Dalamud plugin for FFXIV that adds social-style quests with a native-looking quest tracker.

## Highlights
- FFXIV-style right-side quest tracker
- Daily rotating social quests
- Custom quest popup (accept/decline)
- Chat-based quest progress (emotes and social interactions)

## Links
- GitHub repo: https://github.com/UnbirthdayHatter/SocialMorpho
- Dalamud custom repo URL: https://raw.githubusercontent.com/UnbirthdayHatter/SocialMorpho/main/repo.json

## Install (XIVLauncher)
1. Open `XIVLauncher`.
2. Go to `Settings` -> `Experimental`.
3. Scroll to `Custom Plugin Repositories`.
4. Add:
   `https://raw.githubusercontent.com/UnbirthdayHatter/SocialMorpho/main/repo.json`
5. Open `Plugin Installer`, search for `SocialMorpho`, then install.

## Update
- Updating from Plugin Installer pulls from the same repo URL above.
- If an update looks stale, remove and reinstall the plugin once.

## Commands
- `/morpho`
- `/sm`

## Notes
- Popup quest sound follows the plugin `Sound Enabled` setting.
- Popup and quest icon assets are bundled in release `latest.zip`.
